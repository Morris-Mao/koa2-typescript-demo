# Koa2-Typescript-Demo
This is a demo project with Koa2 + Typescript + Mongoose  
Just Start With 
```shell
npm i --save
npm start
```

Enjoy :)  

## Document Generation
Use Command
```shell
npm run doc
```
To generate document automatically
